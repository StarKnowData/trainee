Python oct() 函数
===============

 [![Python 内置函数](../images/up.gif) Python 内置函数](python-built-in-functions.html)

* * *

描述
--

**oct()** 函数将一个整数转换成8进制字符串。

语法
--

oct 语法：
```
oct(x)
```
参数说明：

*   x -- 整数。

返回值
---

返回8进制字符串。

实例
--

以下实例展示了 oct 的使用方法：
```
>>>oct(10)  '012' 
>>\> oct(20)  '024' 
>>\> oct(15)  '017' 
>>>
```
