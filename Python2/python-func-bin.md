Python bin() 函数
===============

 [![Python 内置函数](../images/up.gif) Python 内置函数](python-built-in-functions.html)

* * *

描述
--

**bin()** 返回一个整数 int 或者长整数 long int 的二进制表示。

### 语法

以下是 bin() 方法的语法:
```
bin(x)
```
### 参数

*   x -- int 或者 long int 数字

### 返回值

字符串。

* * *

实例
--

以下展示了使用 bin 函数的实例：
```
>>>bin(10)  '0b1010' >>\> bin(20)  '0b10100'
```
