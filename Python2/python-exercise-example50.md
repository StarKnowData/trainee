**题目：**输出一个随机数。

**程序分析：**使用 random 模块。

程序源代码：

实例
--
```
#!/usr/bin/python  \# -*- coding: UTF-8 -*-  import  random  #生成 10 到 20 之间的随机数  print  random.uniform(10, 20)
```
以上实例输出结果为：
```
14.4012371357
```
