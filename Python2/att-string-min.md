Python min()方法
==============

 [![Python 字符串](../images/up.gif) Python 字符串](python-strings.html)

* * *

描述
--

Python min() 方法返回字符串中最小的字母。

语法
--

min()方法语法：
```
min(str)
```
参数
--

*   str -- 字符串。

返回值
---

返回字符串中最小的字母。

实例
--

以下实例展示了min()函数的使用方法：
```
#!/usr/bin/python

str = "this-is-real-string-example....wow!!!";
print "Min character: " + min(str);

str = "this-is-a-string-example....wow!!!";
print "Min character: " + min(str);
```
以上实例输出结果如下：
```
Min character: !
Min character: !
```
