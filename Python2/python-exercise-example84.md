**题目：**连接字符串。

**程序分析：**无。
```
实例(Python 2.0+)
---------------

#!/usr/bin/python  \# -*- coding: UTF-8 -*-  delimiter = ','  mylist = \['Brazil', 'Russia', 'India', 'China'\]  print  delimiter.join(mylist)
```
以上实例输出结果为：
```
Brazil,Russia,India,China
```
