**题目：**两个字符串连接程序。

**程序分析：**无。
```
实例(Python 2.0+)
---------------

#!/usr/bin/python  \# -*- coding: UTF-8 -*-  if  \_\_name\_\_ == '\_\_main\_\_': a = "acegikm"  b = "bdfhjlnpq"  \# 连接字符串  c = a \+ b  print  c
```
以上实例输出结果为：
```
acegikmbdfhjlnpq
```
