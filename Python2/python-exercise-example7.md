**题目：**将一个列表的数据复制到另一个列表中。

**程序分析：**使用列表\[:\]。

程序源代码：

实例
--
```
#!/usr/bin/python  
\# -*- coding: UTF-8 -*-  a = \[1, 2, 3\]  b = a\[:\] 
print  b
````
以上实例输出结果为：
```
\[1, 2, 3\]
```
