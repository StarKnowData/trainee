**程序分析：**无。

程序源代码：

实例
--
```
#!/usr/bin/python  \# -*- coding: UTF-8 -*-  i = \['a', 'b'\]  l = \[1, 2\]  print  dict(\[i,l\])
```

```
{'a': 'b', 1: 2}
```
